My Challenges

One of the biggest challenge I have face and keep facing, is building applications that keep the user browsing it, signing up and using it consistently. User retention is quite hard, if you don’t understand the UX concepts, that is why I have enrolled into couple of online courses I try to follow between my lectures and exams.

Additionally, I haven’t worked with that many projects, but I find it hard usually to get accurate location data based on what user was typing, I was not importing packages. I find it hard to implement those functionalities that automatically generate new data based on user input.

For example when the user inputs a car make and you have to fetch queries in the database of all the cars that are currently available and send again a new request based on the selected output.

I had to figure out the complex SQL queries with many tables involved, also how to send the request, Ajax, parsing the JSON result and finally displaying all that neatly. Nowadays, with Laravel I usually just get projects from Composer whenever I can.
